Post-Install process for the package


Congrats! You have installed a brilliant tool to check how much 50 cent is in Russia



First: Give remote site setting access

Go to Setup => Remote Site Settings and fill:

Remote Site Name : Currency

Remote Site URL : http://api.exchangeratesapi.io

Make it active!


Second: Launch a post-install script to launch a scheduled class

Go to Developers Console => Debug => Open Execute Anonymous Window

Execute this code: PostInstallClass.PostInstall();

Good job!


Final: Go and add the 'fifty' component to your home page, record page or an app page.


It's done!
